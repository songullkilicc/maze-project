# Maze Visualization Project

## Overview
This project is a grid-based maze visualization tool built with Python and Pygame. It allows users to interactively create a maze and visualize a pathfinding process.

## Features
- Grid-based interface
- Draw walls using mouse clicks
- Set Start node (press 'S')
- Set End node (press 'E')
- Press SPACE to trigger visualization (animation)

## Technologies Used
- Python
- Pygame

## How to Run
1. Install dependencies:
   pip install pygame

2. Run the program:
   python main.py

## Current Status

- Basic visualization system implemented (with placeholder animation)
